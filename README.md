# EndEffecterBoard2026
## 概要

キャチロボ2026で用いるエンドエフェクタ基板

サーボモータ×3 & エアポンプ×3 & シリアルバスサーボ×1 を駆動可能

CAN, I2C, UARTの通信規格に対応

ABZエンコーダ読み取り

MCU: [STM32G431CBT6](https://www.stmcu.jp/stm32/stm32g4/stm32g4x1/66683/)
